Install W40K Roster Cleaner

Install this app on your device for quick access and offline use

Progressive Web App

Why Install?

Quick Access

Access the app directly from your home screen

Works Offline

Use the app even without an internet connection

Faster Loading

App loads instantly like a native application

Install on iOS (iPhone/iPad)

1

Open Safari

Make sure you're using Safari browser (not Chrome or other browsers)

2

Navigate to the App

Go to the W40K Roster Cleaner website

3

Tap the Share Button

Tap the share button (square with arrow pointing up) at the bottom of the screen

4

Add to Home Screen

Scroll down and tap "Add to Home Screen"

5

Confirm Installation

Tap "Add" to confirm. The app will now appear on your home screen

Install on Android

1

Open Chrome

Use Chrome browser (or any Chromium-based browser like Edge)

2

Navigate to the App

Go to the W40K Roster Cleaner website

3

Open Menu

Tap the three-dot menu button in the top-right corner

4

Install App

Tap "Install app" or "Add to Home screen"

5

Confirm Installation

Tap "Install" to confirm. The app will be added to your home screen

Troubleshooting

Install option not showing?

Make sure you're using the correct browser (Safari on iOS, Chrome on Android) and that you've visited the website at least once.

App not working offline?

Try visiting the website again while online to ensure the app is properly cached for offline use.

Can't find the app after installation?

Check your home screen or app drawer. On some devices, you may need to swipe through multiple home screen pages.
